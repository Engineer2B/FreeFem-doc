## break
Break a loop.
```ffpp
while (condition1){
	...
	if (condition) break;
	...
}
```

## catch
Catch an error, see [try](#try)

## continue
Continue a loop.
```ffpp
for (int i = 0; i < N; ++i){
	...
	if (condition) continue;
	...
}
```

## else
See [if](#if).

## for
For loop.
```ffpp
for (int i = 0; i < N; ++i){
	...
}
```

## if
If condition.
```ffpp
if (condition){
	...
}
else{
	...
}
```

## try
Try a part of code.
```ffpp
try{
	...
}
catch(...){
	...
}
```

## while
While loop.
```ffpp
while (condition){
	...
}
```
